
# AI Article Generation Agent

AI-powered article generation system built using:

- LangChain
- OpenAI GPT
- Tavily Search API
- Streamlit

The application searches the internet for latest information and generates professional SEO-friendly articles using an AI agent workflow.

---

# Features

- AI Article Generation
- Real-Time Internet Search
- OpenAI GPT Integration
- Tavily API Integration
- Professional Streamlit UI
- Agent Activity Tracking

---

# Tech Stack

- Python
- LangChain
- OpenAI
- Tavily

# Setup

## Clone Repository

```bash
git clone https://github.com/palakkk09/ai-article-agent.git
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Create `.env`

```env
OPENAI_API_KEY=your_key
TAVILY_API_KEY=your_key
```

---

# Run Application

```bash
streamlit run app.py
```

---

# Workflow

```text
User Input
↓
Internet Search
↓
AI Processing
↓
Article Generation
↓
Frontend Output based on the type of output choice selected by user 
```
